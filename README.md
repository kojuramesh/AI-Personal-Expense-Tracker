# AI-Powered Personal Expense Tracker

## Overview

This project is being developed as part of the MSIT Capstone Project at the University of the People.

The application helps users manage their personal finances by:

- Track daily expenses
- Set monthly budgets
- Automatically categorize expenses
- Receive AI-powered budgeting recommendations
- View spending analytics through dashboards

## Technologies

- Python
- Flask (planned)
- SQLite
- HTML/CSS/JavaScript
- OpenAI API
- Git & GitHub
